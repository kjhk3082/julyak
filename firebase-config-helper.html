🔥 Firebase 설정 가이드

⚠️ 중요: 제공하신 MongoDB 연결 문자열은 Firebase Firestore가 아닙니다!
Firebase Firestore는 MongoDB와 완전히 다른 서비스입니다.
📌 현재 상황:
- 제공하신 문자열: mongodb://... (MongoDB 형식)
- 필요한 것: Firebase 프로젝트 설정 (웹 앱 구성)

✅ Firebase Console에서 올바른 설정 가져오기

1Firebase Console 접속

브라우저에서 https://console.firebase.google.com 접속

2프로젝트 선택

"julyak" 또는 관련 프로젝트 선택

3프로젝트 설정으로 이동

좌측 상단 ⚙️ 톱니바퀴 → "프로젝트 설정" 클릭

4웹 앱 설정 확인

"일반" 탭 → 하단 "내 앱" 섹션 → 웹 앱 (</> 아이콘)

다음과 같은 형식의 코드가 표시됩니다:

const firebaseConfig = {
  apiKey: "AIzaSy...",
  authDomain: "your-project.firebaseapp.com",
  projectId: "your-project-id",
  storageBucket: "your-project.appspot.com",
  messagingSenderId: "123456789",
  appId: "1:123456789:web:abc123..."
};

5설정값 복사

위 설정값을 복사하여 아래 `.env.local` 형식으로 변환:

NEXT_PUBLIC_FIREBASE_API_KEY=AIzaSy...
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=your-project.firebaseapp.com
NEXT_PUBLIC_FIREBASE_PROJECT_ID=your-project-id
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=your-project.appspot.com
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=123456789
NEXT_PUBLIC_FIREBASE_APP_ID=1:123456789:web:abc123...

🔧 Firestore Database 설정

중요: Firestore Database가 생성되어 있어야 합니다!

1Firestore Database 생성 확인

Firebase Console → "Firestore Database" 클릭

  • "데이터베이스 만들기" 버튼이 보이면 → 데이터베이스 생성 필요
  • 데이터가 보이면 → 이미 생성됨

2데이터베이스 생성 (필요한 경우)

  • "데이터베이스 만들기" 클릭
  • 프로덕션 모드 선택
  • 위치: asia-northeast3 (서울) 선택
  • "사용 설정" 클릭

3보안 규칙 설정

Firestore Database → Rules 탭:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    // 테스트용 임시 규칙 (모든 읽기/쓰기 허용)
    match /{document=**} {
      allow read, write: if true;
    }
  }
}

"게시" 버튼 클릭

🔐 Authentication 설정

1로그인 방법 활성화

Authentication → Sign-in method:

  • ✅ 이메일/비밀번호 사용 설정
  • ✅ Google 사용 설정

2승인된 도메인 추가

Authentication → Settings → Authorized domains:

  • localhost (기본)
  • julyak.jhlab.ai.kr 추가

3테스트 사용자 생성

Authentication → Users → "사용자 추가":

  • 이메일: test@example.com
  • 비밀번호: test123456

💡 MongoDB vs Firebase Firestore

항목 MongoDB Firebase Firestore
연결 방식 Connection String
mongodb://...		 SDK Configuration
firebaseConfig = {...}
인증 Database 사용자/비밀번호 Firebase Auth (OAuth, Email 등)
클라이언트 연결 서버 경유 필요 직접 연결 가능
